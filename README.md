# Artful-Studio
A full-stack website for a creative art studio concept business based in Amman, Jordan. Built as a portfolio project.

## Live Demo
Coming soon — deploying to Vercel

## Project Overview
Artful Studio is a creative platform offering workshops, courses, activities, and an art shop. The website is designed to reflect a warm, playful, and community-driven brand identity.

## Tech Stack
- **Framework** — Next.js 16 + TypeScript
- **Styling** — Tailwind CSS v4
- **CMS** — Sanity (headless)
- **Dev Environment** — GitHub Codespaces
- **Deployment** — Vercel (coming soon)

## Features Built So Far
- [x] Next.js app with App Router

## Getting Started

### Prerequisites
- Node.js 18.17+
- npm

### Run locally

Clone the repository:
```bash
git clone https://github.com/yourusername/Artful-Studio.git
cd Artful-Studio/artful-studio
```

Install dependencies:
```bash
npm install
```

Start Next.js dev server:
```bash
npm run dev
```

Start Sanity Studio:
```bash
cd studio
npx sanity dev
```

Open [http://localhost:3000](http://localhost:3000) for the website
Open [http://localhost:3333](http://localhost:3333) for Sanity Studio
